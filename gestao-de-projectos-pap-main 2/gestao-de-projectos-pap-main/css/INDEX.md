## 🔍 Encontrar Informações

### Preciso de...

#### Cores
→ Consultar **DESIGN-SYSTEM.md** seção "Paleta de Cores"  
→ Ou abrir **COMPONENTS.html** e ver swatches  
→ Ou procurar em **design-system.css** por `--color-`

#### Tipografia
→ Consultar **DESIGN-SYSTEM.md** seção "Tipografia"  
→ Ou ver **COMPONENTS.html** "Typography Demo"  
→ Ou procurar em **design-system.css** por `--font-`

#### Espaçamento
→ Consultar **DESIGN-SYSTEM.md** seção "Espaçamento"  
→ Ou procurar em **design-system.css** por `--spacing-`

#### Componentes
→ Ver **COMPONENTS.html** para demo visual  
→ Copiar HTML de lá direto  
→ Ajustar conforme necessário

#### Responsividade
→ Consultar **DESIGN-SYSTEM.md** seção "Responsividade"  
→ Ver media queries em cada CSS ficheiro  
→ Usar breakpoints: 640px, 768px, 1024px, 1280px

#### Problema/Bug
→ Consultar **README.md** seção "Troubleshooting"  
→ Verificar **PROGRESS.md** se é conhecido  
→ Checar **design-system.css** para variáveis corretas

## ✅ Quick Checklist

Ao criar novo CSS ou componente:

- [ ] Incluir `design-system.css` PRIMEIRO
- [ ] Usar variáveis CSS (não hardcode)
- [ ] Testar em 5 breakpoints (640, 768, 1024, 1280, 1920)
- [ ] Adicionar focus states
- [ ] Validar contraste de cores (WCAG AA)
- [ ] Usar espaçamento 8px grid
- [ ] Documentar em COMPONENTS.html
- [ ] Manter nomes semânticos

## 📚 Referência Rápida

### Cores Principais
```css
--color-primary-600:   #2563eb (Azul)
--color-success-600:   #16a34a (Verde)
--color-warning-600:   #ea580c (Laranja)
--color-danger-600:    #dc2626 (Vermelho)
```

### Espaçamento Comum
```css
--spacing-2:  8px   (inputs)
--spacing-4:  16px  (padding padrão)
--spacing-6:  24px  (card padding)
--spacing-8:  32px  (section padding)
```

### Tipografia Comum
```css
--font-size-xs:   12px
--font-size-sm:   14px
--font-size-base: 16px (padrão)
--font-size-lg:   18px
--font-size-xl:   20px
```

### Breakpoints
```css
640px   /* Mobile/Tablet pequena */
768px   /* Tablet */
1024px  /* Desktop */
1280px  /* Desktop grande */
```
