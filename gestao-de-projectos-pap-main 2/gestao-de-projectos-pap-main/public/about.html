Sobre o Sistema

Sistema de Gestão de Projectos Académicos para o Instituto Técnico Médio Privado Santa Ana & Noesa

Objectivo

Centralizar os projectos defendidos e facilitar a pesquisa, gestão e acompanhamento por parte dos orientadores e administradores académicos.

Funcionalidades Principais

  • Listagem e filtros avançados de projectos
  • Detalhes completos e download de documentos
  • Área administrativa para gestão de dados

Segurança

Autenticação segura com sistema de controle de acesso baseado em papéis e proteção de dados sensíveis.

Tecnologias Utilizadas

Frontend

HTML5, CSS3, JavaScript, Bootstrap 5

Backend

PHP 7+, MySQL, RESTful APIs

Infraestrutura

Servidor Web, Sistema de Ficheiros, Backups
